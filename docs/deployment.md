# 🌐 Deployment

## Modes

### SaaS
- No setup required
- Limited features

### Self-Hosted
- Full feature access
- Project generation + downloads

---

## 🔍 Feature Comparison: Self-Hosted vs SaaS

Magdi-AI supports NLP-driven test generation with multiple input methods — including natural language, URLs, uploaded files, and screen scraping — enabling users to go from instruction to execution in minutes.

| Feature / Capability                          | Self-Hosted | SaaS  |
|-----------------------------------------------|-------------|-------|
| MagdiPartner (Pair Programming)               | ✅ Yes      | ✅ Yes |
| MagdiAnalyzer (Test Results Analysis)         | ✅ Yes      | ✅ Yes |
| MagdiWriter (QA Documentation)                | ✅ Yes      | ✅ Yes |
| MagdiSage (Research & General Assistance)     | ✅ Yes      | ✅ Yes |
| MagdiCreator (NLP Test Generation Wizard)     | ✅ Yes      | ❌ No  |
| Natural Language Test Generation (NLP)        | ✅ Yes      | ❌ No  |
| Multi-Input Instructions (Text, URL, Files, Scraping) | ✅ Yes | ❌ No |
| Create New Automation Frameworks              | ✅ Yes      | ❌ No  |
| Update Existing Automation Projects           | ✅ Yes      | ❌ No  |
| Screen Scrape Engine (Depth-Based Crawling)   | ✅ Yes      | ❌ No  |
| Multi-Tool Support (Playwright, Cypress, Selenium, Postman, JMeter) | ✅ Yes | ❌ No |
| Mobile & Multi-Platform Test Generation       | ✅ Yes      | ❌ No  |
| Smart Tool + Framework Auto-Selection         | ✅ Yes      | ❌ No  |
| Execution-Ready Test Projects (Run Immediately) | ✅ Yes    | ❌ No  |
| Download Generated Project Bundles            | ✅ Yes      | ❌ No  |
| Local File System Access                      | ✅ Yes      | ❌ No  |
| Zero Setup / Hosted Experience                | ❌ No       | ✅ Yes |

ℹ️ **Note:**  

SaaS mode is designed for exploration, analysis, and collaboration.  
Self-Hosted mode unlocks full NLP-driven test generation, multi-input workflows, execution-ready projects, and full environment control.
