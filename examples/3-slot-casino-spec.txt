Endpoint: POST /spin
Request
{
  "user_id": "string"
}
Response
{
  "reels": ["🍒", "🍋", "⭐"],
  "result": "win" | "lose",
  "payout": number,
  "new_balance": number
}

Apply payout only on win
Return deterministic structure
